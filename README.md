# HomeScout

AI-powered home search that scores, ranks, and explains listings based on your preferences.

## Stack
- Cloudflare Pages (frontend)
- Cloudflare Workers (backend + cron)
- Cloudflare Workers AI (Llama/Mistral for scoring + summaries)
- Cloudflare D1 (SQLite database)
- Cloudflare Browser Rendering (scraping)
- Resend (email notifications)

## Status
MVP in progress.
